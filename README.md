# Repo-skills
Este repositório tem a intenção de armazenar todas skills necessárias pra criar ou desenvolver um projeto backend Java.

## Skills disponíveis

As skills estão organizadas em `.agents/skills/` e cobrem os principais tópicos para desenvolvimento backend Java com Spring Boot:

| Skill | Descrição |
|-------|-----------|
| [spring-boot-setup](/.agents/skills/spring-boot-setup.md) | Configuração inicial de um projeto Spring Boot |
| [rest-api-development](/.agents/skills/rest-api-development.md) | Desenvolvimento de APIs RESTful com Spring MVC |
| [jpa-database-configuration](/.agents/skills/jpa-database-configuration.md) | Persistência com Spring Data JPA e Flyway |
| [testing-junit-spring](/.agents/skills/testing-junit-spring.md) | Testes unitários e de integração com JUnit 5 e Testcontainers |
| [maven-build-configuration](/.agents/skills/maven-build-configuration.md) | Configuração de build com Maven |
| [spring-security-jwt](/.agents/skills/spring-security-jwt.md) | Autenticação e autorização com Spring Security e JWT |
