# Skill: @criar_projeto_hexagonal

## Objetivo
Criar projeto seguindo arquitetura hexagonal

## Regras
- Domínio não depende de framework
- Separar ports e adapters

## Execução
1. Criar base do projeto
2. Aplicar estrutura:

/domain
/application
/adapter/in
/adapter/out

## Regra de ouro
Domínio é independente
